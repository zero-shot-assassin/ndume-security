## Brand Photography Notes

- **Colour palette:** All images should feature dark navy blues (`#0d1b4b`) and gold accents (`#c9a84c`)
- **Tone:** Professional, authoritative, trustworthy — never aggressive or militaristic
- **Setting:** Always Kenyan/Nairobi context — recognisable architecture, African faces, local environments
- **Resolution:** Minimum 1600×900px for hero images; 800×600px for card images
- **Format:** WEBP preferred for web (smaller file size); fallback PNG
